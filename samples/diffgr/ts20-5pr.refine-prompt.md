# DiffGR 仮想PR分割のブラッシュアップ依頼

目的: 1つの大きい差分を、仮想PR(グループ)に分割してレビューしやすくする。
制約:
- 全チャンクを必ずどれか1つのグループに割り当てる
- グループ名は日本語（短く、目的が伝わる）
- 各グループはレビュー可能な粒度（大きすぎたら分割、小さすぎたら統合）

出力フォーマット（必須）: 次のJSONだけを返してください。
```json
{ "rename": { "g-id": "新しい名前" }, "move": [ { "chunk": "chunk_id", "to": "g-id" } ] }
```
